Mantisnet Cooking

Mantisnet Cooking es una aplicación web de recetas que permite a los usuarios visualizar recetas, y a los administradores agregarlas, editarlas y eliminarlas, con soporte para subir imágenes.

Utiliza una base de datos PostgreSQL en Supabase y Cloudinary para la gestión de imágenes.

Características:
  • Visualización y gestión de recetas.
  • Autenticación de administradores.
  • Carga de imágenes de forma eficiente.
Tecnologías utilizadas:
  • Node.js
  • Express
  • PostgreSQL
  • Supabase
  • Multer
  • Cloudinary

Aplicación Gestion de Torneos

Esta aplicación permite gestionar torneos de fútbol, facilitando la organización de equipos, partidos y resultados.

Con un enfoque en la facilidad de uso, los administradores pueden crear torneos, agregar equipos y programar partidos.

Características:
  • Gestión de equipos y torneos.
  • Programación de partidos y seguimiento de resultados.
  • Interfaz intuitiva para usuarios y administradores.
Tecnologías utilizadas:
  • Python
  • Flask
  • PostgreSQL
  • React
